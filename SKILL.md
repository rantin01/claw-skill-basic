# Skill Name
ClawCredit Explanation Skill

## Description
Skill ini membantu agent menjelaskan ClawCredit kepada pengguna
dengan bahasa sederhana dan mudah dipahami.

## What This Skill Does
- Menjelaskan apa itu ClawCredit
- Menjelaskan kegunaan ClawCredit
- Menjawab pertanyaan dasar pengguna
- Memberikan informasi tanpa klaim keuntungan

## What This Skill Does NOT Do
- Tidak menjanjikan reward
- Tidak menjamin hasil
- Tidak memberikan saran finansial

## Usage Rules
- Gunakan bahasa netral dan edukatif
- Jika informasi tidak diketahui, katakan dengan jujur
- Ikuti semua aturan dari sistem atau skill lain

## Response Style
- Singkat
- Jelas
- Ramah

## Disclaimer
This skill is for informational purposes only.
